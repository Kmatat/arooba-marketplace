## ğŸ’° Business Model

### The Additive Uplift Model (Cost-Plus)
Arooba **never reduces** vendor prices. Revenue is generated by adding service fees **on top** of the vendor's base price.

```
Final Price = Vendor Base Price + Cooperative Fee (5%, if applicable) + Marketplace Uplift
```

### Revenue Streams
1. Platform service fee per order (3â€“5 EGP)
2. Vendor subscription plans (Phase 2: 150â€“300 EGP/month)
3. Featured listings & promotions
4. Institutional/bulk orders (schools, hotels)
5. In-platform advertising
